# CI/CD Demo App

Demo Node.js application for practicing DevSecOps CI/CD pipelines using:

- Jenkins
- Docker
- SonarQube
- Trivy
- Jest
- GitHub

## Run Locally

```bash
cd app
npm install
npm start
```

## Run Tests

```bash
npm test
```

## Build Docker Image

```bash
docker build -t cicd-demo .
```
